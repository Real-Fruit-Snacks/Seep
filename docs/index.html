FEATURES

16 checks, zero disk footprint

Fileless PowerShell agent with 16 enumeration checks. 97-tool catalog with SHA256 verification. MITRE ATT&CK-mapped recommendations.

Fileless Agent

Agent runs entirely in memory via IEX cradle. Results are AES-256-CBC encrypted, compressed, and uploaded over HTTP. AMSI/ETW/Script Block Logging bypassed automatically.

ARCHITECTURE

Two-component architecture

Python CLI on the attack box orchestrates everything. Composed PowerShell agent runs on the target and reports back.

1

CLI Server

Click-style CLI with init, serve, catalog, compose, report commands. Unified HTTP handler for agent delivery, tool serving, and result upload.

2

Agent Composer

16 PowerShell check modules with metadata headers. Cherry-pick checks with --checks/--exclude. CSPRNG identifier randomization with --obfuscate.

3

Tool Catalog

97 tool definitions in YAML with SHA256 hashes, categories, and MITRE triggers. Download, verify integrity, and serve via the HTTP server.

4

Report Generator

Finding-to-tool mapping with MITRE ATT&CK recommendations. HTML, Markdown, and JSON report generation with executive summary and remediation guidance.

GETTING STARTED

Install and enumerate

Python 3.9+ on the attack box, PowerShell 3.0+ on the target. Install with pipx and initialize a workspace.

install.sh
$ git clone https://github.com/Real-Fruit-Snacks/Seep.git
$ cd Seep && pipx install .

# Initialize workspace and start server
$ seep init --workdir /tmp/op1
$ seep serve --workdir /tmp/op1
HTTP :80 | Upload :8000 | Token: a8f3c2
usage.sh
# Compose minimal agent
$ seep compose --checks system_info,quick_wins

# Download tool catalog
$ seep catalog download --all --workdir /tmp/op1

# Generate HTML report
$ seep report results.json --format html -o report.html

# Verify tool integrity
$ seep catalog verify --workdir /tmp/op1
DETECTION AWARENESS

Know the boundaries

Evades

  • AMSI — bypass in cradle before agent loads
  • ETW — disabled at runtime, Script Block Logging off
  • Disk forensics — fileless execution via IEX cradle
  • IIS fingerprinting — server header spoofing
  • Auth probing — token-gated endpoints return 404

Visible To

  • Advanced EDR — behavioral and kernel-level detection
  • Memory scanning — PowerShell in-memory artifacts
  • Network monitoring — HTTP connection metadata
  • PowerShell v5+ — deep script inspection
  • Defender ATP — cloud-connected analysis
